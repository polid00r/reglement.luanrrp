📜 Règlement LunarRP

Règlement Général Règlement Discord Règlement Streaming Règlement Illégal Règlement Légal Sanctions Lexique

📜 Règlement Général

2.01 - Il est interdit de vendre vos possessions en jeu en utilisant de la vraie argent.

2.02 - Le RP sexe est totalement interdit.

2.03 - L'utilisation d'un sac est obligatoire pour tout objet de grande dimension.

2.04 - Les cheats/mod-menus sont interdits sous peine de bannissement.

2.05 - Une scène RP doit être jouée jusqu'au bout avant de faire une réclamation.

💬 Règlement Discord

1.01 - Aucun comportement discriminatoire ou harcelant ne sera toléré.

1.02 - Il est interdit de partager des contenus illégaux ou inappropriés.

1.03 - Le spam est interdit.

1.04 - Le non-respect de ces règles peut entraîner des sanctions.

🎥 Règlement Streaming

6.01 - Le streamhack est interdit.

6.02 - La modération peut utiliser les streams comme preuve.

6.03 - Un salon "stream" est disponible sur Discord pour les streamers.

6.04 - Il est interdit de diffuser les points chauds (drogues, blanchiment).

6.05 - Les clips peuvent être utilisés pour des compilations YouTube/TikTok non-monétisées.

🔫 Règlement Illégal

4.01 - Les otages ne doivent pas être des joueurs consentants ou des faux otages.

4.02 - Les descentes et guerres doivent être validées via un GDOC.

4.03 - Un gang doit respecter la cohérence RP et avoir peur des organisations plus puissantes.

⚖️ Règlement Légal

5.01 - Les forces de l'ordre doivent respecter leurs protocoles.

5.02 - Il est interdit de se faire passer pour un agent si vous ne l'êtes pas.

5.03 - Un patron d’entreprise doit payer ses employés chaque semaine.

🚨 Sanctions

3.01 - Les sanctions sont décidées selon la charte en place.

3.02 - Pour contester une sanction, ouvrez un ticket sur Discord.

3.03 - Un ban permanent peut être discuté avec l'administration.

📖 Lexique

Metagaming : Utilisation d'informations HRP en RP.

Powergaming : Actions irréalistes en jeu.

No Pain : Ne pas jouer la douleur ressentie.

WinRP : Ne pas prendre en compte l’environnement RP.

Usebug : Exploiter des bugs pour un avantage.
